# Meets
Este é a parte Web do Projeto Meets (antigo Fatec Meets) que nesta versão está em PHP

é possivel acessar esta versão no navegador através deste link
http://meets.rf.gd/
(caso o navegador indique "página insegura" ou "página não encontrada", basta retornar uma página e tentar acessar o link novamente)

*INSTALAÇÃO*

1.Para instalar e testar o projeto de forma local basta possuir o XAMPP ou WAMP instalado em sua maquina,
2.extraia o arquivo zip dentro da pasta htdocs do xampp (ou correlata do wamp)
3.ligue o apache e Mysql no xampp
4.No phpmyadmin adicione o arquivo SQL "if0_38701439_fatecmeets.sql" que esta na pasta off
5.Na barra de pesquisa superior do navegador digite "localhost/Meets_Web"
6.Testar o Projeto
